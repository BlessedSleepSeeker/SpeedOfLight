# Bubbly Bold
Bubbly Bold is a 16pt pixel art font designed to have the visual effect of bold pixel bubble letters.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 16px
- Scales: 16, 32, 48, 64, etc
- Ascender Max: 9px
- Descender Max: 3px
